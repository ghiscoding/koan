Full stack JavaScript Web development boilerplate.

Koa framework provides the RESTful backend API, utilizing the harmony generators.
Aurelia is our fully static client side app framework.
Node.js is super glue, runtime for the backend, build system for the front.
Finally, MongoDB is our NoSQL database of choice in this stack and sample app.
